switch case中强烈建议使用break，不然容易导致错误

不合规示例

switch (myVariable) {
  case 1:
    foo();
    break;
  case 2:  // case 2将执行 'doSomething()' 和 'doSomethingElse()' 两个方法，而这是你的真实意图吗？
    doSomething();
  default:
    doSomethingElse();
    break;
}

合规示例

switch (myVariable) {
  case 1:
    foo();
    break;
  case 2:
    doSomething();
    break;
  default:
    doSomethingElse();
    break;
}

例外

在以下情况下，此规则将被放宽：

switch (myVariable) {
  case 0:                                //空置的 case 0，与case 1相同的行为。
  case 1:
    doSomething();
    break;
  case 2:                                // 使用 return 语句
    return;
  case 3:
    console.log("this case falls through")
  case 4:                                // 使用 throw 语句
    throw new IllegalStateException();
  case 5:                                // 使用 continue 语句
    continue;
  default:                               // 对于default，break是可选的
    doSomethingElse();
}
