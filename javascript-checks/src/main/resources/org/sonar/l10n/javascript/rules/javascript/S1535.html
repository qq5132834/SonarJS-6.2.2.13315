for...in 语句允许您遍历一个对象的所有属性的名称。

因此，每个 for...in 语句的主体都应该包含在一个 if 语句中，该语句过滤对哪些属性进行操作。 它可以选择特定类型或值范围，也可以排除函数，或者从原型中排除属性。

不合规示例

for (name in object) {
    doSomething(name);  // 不合规
}

合规示例

for (name in object) {
  if (object.hasOwnProperty(name)) {
    doSomething(name);
  }
}

例外

用于克隆对象的循环将被忽略。

for (prop in obj) {
  a[prop] = obj[prop];  // 例外合规
}
