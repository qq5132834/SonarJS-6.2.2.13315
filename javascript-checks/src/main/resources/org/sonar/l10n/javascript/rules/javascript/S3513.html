JavaScript 的神奇之处在于，可以将参数传递给不声明参数的函数，另一方面，也可以在无参数 function 中使用那些传入的参数。

但仅仅因为你可以，并不意味着你应该。 在没有显式声明的函数中使用参数会让代码维护人员感到困惑。

如果您不想显式命名参数，请使用 ... 语法指定需要可变数量的参数。 然后在函数内部，您将处理一流的数组，而不是类似数组的结构。

不合规示例

function concatenate() {
  let args = Array.prototype.slice.call(arguments);  // 不合规
  return args.join(', ');
}

function doSomething(isTrue) {
  var args = Array.prototype.slice.call(arguments, 1); // 不合规
  if (!isTrue) {
    for (var arg of args) {
      ...
    }
  }
}

合规示例

function concatenate(...args) {
  return args.join(', ');
}

function doSomething(isTrue, ...values) {
  if (!isTrue) {
    for (var value of values) {
      ...
    }
  }
}
