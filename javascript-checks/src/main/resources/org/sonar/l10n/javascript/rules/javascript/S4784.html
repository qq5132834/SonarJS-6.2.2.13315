使用正则表达式是安全敏感的。 它在过去导致了以下漏洞：

  • CVE-2017-16021
  • CVE-2018-13863

敏感示例

const regex = /(a+)+b/; // 敏感
const regex2 = new RegExp("(a+)+b"); // 敏感

str.search("(a+)+b"); // 敏感
str.match("(a+)+b"); // 敏感
str.split("(a+)+b"); // 敏感
