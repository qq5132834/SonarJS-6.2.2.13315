switch中default子句应该在最后

不合规示例

switch (param) {
  case 0:
    doSomething();
    break;
  default: // default 子句应该在最后
    error();
    break;
  case 1:
    doSomethingElse();
    break;
}

合规示例

switch (param) {
  case 0:
    doSomething();
    break;
  case 1:
    doSomethingElse();
    break;
  default:
    error();
    break;
}
