跳转语句（return、break 和continue）和throw 表达式将控制流移出当前代码块。 因此，跳转后的任何语句都是死代码。

不合规示例

function fun(a) {
  var i = 10;
  return i + a;
  i++;             // 不合规; 这里无法执行到
}

合规示例

function fun(int a) {
  var i = 10;
  return i + a;
}

例外

此规则忽略 switch 子句中无法访问的 break 语句。

switch (x) {
  case 42:
     return 43;
     break;   // 合规
  default:
    doSomething();
}
