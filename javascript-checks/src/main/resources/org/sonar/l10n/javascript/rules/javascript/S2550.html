当将Backbone.js框架与包含数组或对象的模型defaults一起使用时，defaults应定义为函数而不是对象。 这是因为对象和数组在JavaScript中是通过引用传递的。 因此，包含数组或对象的defaults对象将设置每个实例的默认值，以指向相同的共享对象或数组。

改用函数，将为每个实例剥离对象或数组的新副本。

不合规示例

var Person = Backbone.Model.extend({
    defaults: {  // 不合规；Person的每个实例将共享相同的favoriteColors实例
        favoriteColors: ["blue","purple","raspberry"]
    }
});

合规示例

var Person = Backbone.Model.extend({
    defaults: function() {
      return {
        favoriteColors: ["blue","purple","raspberry"]
      };
    }
});
