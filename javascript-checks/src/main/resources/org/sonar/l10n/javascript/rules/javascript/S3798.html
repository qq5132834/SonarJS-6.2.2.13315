在全局范围中声明的任何变量或函数都会隐式地附加到全局对象（浏览器环境中的window对象）。 要使用window显式得将该变量或函数添加到全局对象中。 当它仅用于本地时，应使用const或let关键字（自ECMAScript 2015以来）或在立即调用函数表达式（IIFE）中声明。

使用模块时不应激活此规则。

不合规示例

var myVar = 42;       // 不合规
function myFunc() { } // 不合规

合适示例

window.myVar = 42;
window.myFunc = function() { };

或者

let myVar = 42;
let myFunc = function() { }

或者

// IIFE
(function() {
  var myVar = 42;
  function myFunc() { }
})();
