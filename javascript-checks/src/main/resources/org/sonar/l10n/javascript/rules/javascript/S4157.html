没有理由重复默认类型，除非它在列表的前面并且其他非默认类型紧随其后。 取而代之的是，当它不是默认值时，将其排除在外并仅提供类型。

不合规示例

function foo<N = number, S = string>() {}
foo<number, string>();  // 不合规； 两种类型都是冗余的
foo<string, string>();  // 不合规； 第二个字符串是多余的
foo<number, number>();  // 忽略； 号码是多余的，但需要

合规示例

function foo<N = number, S = string>() {}
foo();
foo<string>();
foo<number, number>();
