任何语句或语句块都可以通过标签来标识， 但这些标签只能用于 while、do-while、for 和 switch 语句。 在任何其他上下文中使用标签会导致非结构化、混乱的代码。

不合规示例

myLabel: if (i % 2 == 0) {  // 不合规
  if (i == 12) {
    console.log("12");
    break myLabel;
  }
  console.log("Odd number, but not 12");
}

合规示例

myLabel: for (i = 0; i < 10; i++) {   // 合规
  console.log("Loop");
  break myLabel;
}
