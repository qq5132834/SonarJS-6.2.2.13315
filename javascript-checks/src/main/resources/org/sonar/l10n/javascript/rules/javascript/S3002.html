一元运算符 + 和 - 可用于将某些值类型转换为数值。 但并不是每个值都可以转换为 Number 类型； 将它与对象一起使用，结果将是 NaN。 这可能会让维护者感到困惑。

不合规示例

var obj = {x : 1};
doSomethingWithNumber(+obj);    // 不合规

function foo(){
  return 1;
}
doSomethingWithNumber(-foo);    // 不合规

合规示例

var obj = {x : 1};
doSomethingWithNumber(+obj.x);

function foo(){
  return 1;
}
doSomethingWithNumber(-foo());

var str = '42';
doSomethingWithNumber(+str);
