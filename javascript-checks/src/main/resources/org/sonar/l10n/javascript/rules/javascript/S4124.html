在 interface 中声明一个构造函数，您将获得一个名为 constructor 的简单方法。 如果在 interface 中创建一个 new 方法，也会发生同样的事情：您将获得一个名为 new 的简单方法。

相反，其意图可能是指定该类型并非源自 TypeScript 文件。 在这种情况下，只需使用 declare class 语法即可。

不合规示例

interface TypeDeclaredElsewhere {
  someMethod(): number;
  new(b: boolean): TypeDeclaredElsewhere; // 不合规
  constructor(b: boolean): void; // 不合规
}

合规示例

declare class TypeDeclaredElsewhere {
  someMethod(): number;
  constructor(b: boolean);
}
