JavaScript变量范围可能特别难以理解。假如，在函数或for循环的子句中声明变量而不使用let, const or var关键字时发生的情况。

let和const在ECMAScript 2015中引入，现在是变量声明的首选关键字。

不合规示例

function f(){
  i = 1;         // 不合规；i是全局

  for (j = 0; j < array.length; j++) {  // 不合规；j现在也是全局性的
    // ...
  }
}

合规示例

function f(){
  var i = 1;

  for (let j = 0; j < array.length; j++) {
    // ...
  }
}
