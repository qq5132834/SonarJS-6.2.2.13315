联合类型不应该有太多元素

不合规示例

默认阈值为 3：

let x: MyType1 | MyType2 | MyType3 | MyType4; // 不合规

function foo(p1: string, p2: MyType1 | MyType2 | MyType3 | MyType4) { // 不合规
    // ...
}

合规示例

type MyUnionType = MyType1 | MyType2 | MyType3 | MyType4; // 合规，type声明被忽略
let x: MyUnionType;

function foo(value: string, padding: MyUnionType) {
    // ...
}
