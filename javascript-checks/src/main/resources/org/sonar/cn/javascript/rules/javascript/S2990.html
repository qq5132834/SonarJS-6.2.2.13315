在对象外部使用关键字this时，指的是全局的this对象，与window对象是同一个网页。 这样的用途可能会让维护者感到困惑。 相反，只需删除 this，或将其替换为 window； 它将具有相同的效果并且更具可读性。

不合规示例

this.foo = 1;   // 不合规
console.log(this.foo); // 不合规

function MyObj() {
  this.foo = 1;
}

MyObj.func1 = function() {
  if (this.foo == 1) {
    // ...
  }
}

合规示例

foo = 1;
console.log(foo);

function MyObj() {
  this.foo = 1;
}

MyObj.func1 = function() {
  if (this.foo == 1) {
    // ...
  }
}
