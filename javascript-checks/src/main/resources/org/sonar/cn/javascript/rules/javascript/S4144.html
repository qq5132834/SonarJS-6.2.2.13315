当两个函数具有相同的实现时，要么有意为之，要么是误操作。这种情况建议重构代码。

不合规示例

function calculateCode() {
  doTheThing();
  doOtherThing();
  return code;
}

function getName() {  // 不合规
  doTheThing();
  doOtherThing();
  return code;
}

合规示例

function calculateCode() {
  doTheThing();
  doOtherThing();
  return code;
}

function getName() {
  return calculateCode();
}

例外

F少于 3 行的函数将被忽略。
