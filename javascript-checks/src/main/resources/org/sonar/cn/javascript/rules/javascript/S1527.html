以下单词可能会在该语言的未来发展中用作关键字，因此应避免将它们用作标识符，以便更容易采用这些潜在的未来版本：

  • await
  • class
  • const
  • enum
  • export
  • extends
  • implements
  • import
  • interface
  • let
  • package
  • private
  • protected
  • public
  • static
  • super
  • yield

不合规示例

var package = document.getElementsByName("foo"); // 不合规
var someData = { package: true };                 // 合规, 因为它在这里不用作标识符

Compliant Solution

var elements = document.getElementsByName("foo"); // 合规
