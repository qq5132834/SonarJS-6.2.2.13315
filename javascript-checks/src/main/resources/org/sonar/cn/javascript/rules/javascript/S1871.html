在cases语句中有两个cases，或者在if-else中有两条分支具有相同的实现，最好是重复代码，最坏是编码错误。

不合规示例

switch (i) {
  case 1:
    doFirstThing();
    doSomething();
    break;
  case 2:
    doSomethingDifferent();
    break;
  case 3:  // 不合规；重复case 1的实施
    doFirstThing();
    doSomething();
    break;
  default:
    doTheRest();
}

if (a >= 0 && a < 10) {
  doFirstThing();
  doTheThing();
}
else if (a >= 10 && a < 20) {
  doTheOtherThing();
}
else if (a >= 20 && a < 50) {
  doFirstThing();
  doTheThing();  // 不合规；重复第一个条件
}
else {
  doTheRest();
}
