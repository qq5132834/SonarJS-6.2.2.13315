当函数调用中参数的名称与函数参数的名称匹配时，这有助于编写更清晰、更可读的代码。 但是，如果名称匹配，但传递的顺序与函数参数不同，则表示参数顺序错误，这可能会导致意外结果。

不合规示例

function divide(divisor, dividend) {
  return divisor/dividend;
}

function doTheThing() {
  var divisor = 15;
  var dividend = 5;

  var result = divide(dividend, divisor);  // 不合规；操作成功，但结果意外
  //...
}

合规示例

function divide(divisor, dividend) {
  return divisor/dividend;
}

function doTheThing() {
  var divisor = 15;
  var dividend = 5;

  var result = divide(divisor, dividend);
  //...
}
