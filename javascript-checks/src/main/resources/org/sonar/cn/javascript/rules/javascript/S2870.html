delete运算符可用于从任何对象中删除属性。 数组是对象，因此这里也可以使用delete运算符，但如果是，数组中会留下一个洞，因为索引/键不会被移动以反映删除。

不合规示例

var myArray = ['a', 'b', 'c', 'd'];

delete myArray[2];  // 不合规。 myArray => ['a', 'b', undefined, 'd']
console.log(myArray[2]); // 预期值为 d，但实际上输出未定义

合规示例

var myArray = ['a', 'b', 'c', 'd'];

// 从index为2的地方移除一个元素
removed = myArray.splice(2, 1);  // myArray => ['a', 'b', 'd']
console.log(myArray[2]); // 输出 'd'
