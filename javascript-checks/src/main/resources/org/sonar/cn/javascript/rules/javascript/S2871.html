无论数组中的类型如何，默认排序顺序都是字母，而不是数字。 具体来说，即使一个数组只包含数字，它中的所有值都将转换为字符串，并按如下顺序按字典排序：1、15、2、20、5。

幸运的是，sort方法允许您传递一个可选的比较函数来指定排序顺序。 当提供比较函数时，返回的顺序取决于比较函数的返回值。

不合规示例

var myarray = [80, 3, 9, 34, 23, 5, 1];

myarray.sort();
console.log(myarray); // 输出: [1, 23, 3, 34, 5, 80, 9]

合规示例

var myarray = [80, 3, 9, 34, 23, 5, 1];

myarray.sort((a, b) => (a - b));
console.log(myarray); // 输出: [1, 3,  5, 9, 23, 34, 80]
