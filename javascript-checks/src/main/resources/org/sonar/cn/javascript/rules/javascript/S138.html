一个函数如果过大，意味着过多的功能。这样的功能不可避免地变得难以理解，因此难以维护。

在特定阈值以上，强烈建议将其重构为专注于定义良好的任务较小的函数。

这些较小的函数不仅更容易理解，而且可能更容易测试。

例外

此函数忽略立即调用函数表达式（IIFE），后者是创建和调用的函数，而从未指定名称。

(function () { // 此规则忽略

  function open() {  // 经典函数声明；未被忽略
    // ...
  }

  function read() {
    // ...
  }

  function readlines() {
    // ...
  }
})();
