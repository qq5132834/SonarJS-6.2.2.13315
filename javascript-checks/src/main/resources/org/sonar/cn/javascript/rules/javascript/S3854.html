存在必须调用 super() 的情况和不能调用 super() 的情况。

基本规则是：非派生类中的构造函数不能调用 super()； 派生类中的构造函数必须调用 super()。

不合规示例

class Dog extends Animal {
  constructor(name) {
    super();
    this.name = name;
    super();         // 不合规
    super.doSomething();
  }
}

合规示例

class Dog extends Animal {
  constructor(name) {
    super();
    this.name = name;
    super.doSomething();
  }
}
