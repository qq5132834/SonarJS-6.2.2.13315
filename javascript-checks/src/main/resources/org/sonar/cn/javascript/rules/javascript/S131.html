"switch" 语句中应该有 "default" 子句

不合规代码

switch (param) {  //未使用default子句
  case 0:
    doSomething();
    break;
  case 1:
    doSomethingElse();
    break;
}

switch (param) {
  default: // default子句应该在最后
    error();
    break;
  case 0:
    doSomething();
    break;
  case 1:
    doSomethingElse();
    break;
}

合规示例

switch (param) {
  case 0:
    doSomething();
    break;
  case 1:
    doSomethingElse();
    break;
  default:
    error();
    break;
}

  • MITRE, CWE-478 - Switch 中缺少 Default
