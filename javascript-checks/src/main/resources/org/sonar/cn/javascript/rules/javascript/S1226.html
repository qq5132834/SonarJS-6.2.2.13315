虽然从技术上讲，从函数体中分配给参数是正确的，但这会降低代码的可读性， 因为开发人员无法在不经过整个函数的情况下判断是否正在访问原始参数或某个临时变量。 此外，一些开发人员可能还希望函数参数的赋值对调用者可见，但事实并非如此，这种可见性的缺乏可能会使他们感到困惑。 相反，所有参数、捕获的异常和foreach参数都应视为常量。

不合规示例

function MyClass(name, strings) {
  name = foo;                    // 不合规

  for (var str of strings) {
    str = "";  // 不合规
  }
}
