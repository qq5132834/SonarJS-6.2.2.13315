Internet Explorer 提供了一种使用条件注释（由注释中的 @cc_on 语句激活）在运行时更改 JavaScript 代码的方法。 使用此预处理功能会降低可读性和可维护性，并且会妨碍自动化工具。 而且，它是特定于 Internet Explorer 的，不适用于其他浏览器。

大多数时候，可以通过一些重构轻松避免使用这些条件注释——使用现代跨浏览器的 JavaScript 框架和库。

不合规示例

/*@cc_on
  @if (@_jscript_version >= 5.5)
    document.write("You are using IE5.5 or newer");
  @else
    document.write("You are using IE5 or older");
  @end
  @*/
