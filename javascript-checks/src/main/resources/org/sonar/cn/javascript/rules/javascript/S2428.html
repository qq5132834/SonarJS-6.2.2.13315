在对象声明中初始化对象的属性，比其他方法更清晰：创建一个空对象，然后逐个赋值。

当满足以下模式时，会出现问题：

  • 将创建一个空对象。
  • 连续的单行语句将向创建的对象添加属性。

不合规示例

let person = {};  // 不合规
person.firstName = "John";
person.middleInitial = "Q";
person.lastName = "Public";

合规示例

let person = {
  firstName: "John",
  middleInitial: "Q",
  lastName: "Public",
}
