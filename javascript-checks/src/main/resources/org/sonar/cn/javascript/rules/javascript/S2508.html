使用Backbone.js框架时，模型属性的名称不应包含空格。 这是因为Events对象接受以空格分隔的事件列表，因此名称中带有空格的属性可能会被误解。

不合规示例

Person = Backbone.Model.extend({
        defaults: {
            'first name': 'Bob',      // 不合规
            'birth date': new Date()  // 不合规
        },
    });

合规示例

Person = Backbone.Model.extend({
        defaults: {
            firstName: 'Bob',
            birthDate: new Date()
        },
    });
