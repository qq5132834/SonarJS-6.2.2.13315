switch 语句不应包含标签，这可能导致错误结果。

不合规示例

例1

switch (day) {
  case MONDAY:
  case TUESDAY:
  WEDNESDAY:   // 替换为 "case WEDNESDAY"
    doSomething();
    break;
  ...
}

例2

switch (day) {
  case MONDAY:
    break;
  case TUESDAY:
    foo:for(i = 0 ; i < X ; i++) {
         /* ... */
        break foo;  // 这里的 break 语句与 case TUESDAY 无关
         /* ... */
    }
    break;
    /* ... */
}

合规示例

例1

switch (day) {
  case MONDAY:
  case TUESDAY:
  case WEDNESDAY:
    doSomething();
    break;
  ...
}

例2

switch (day) {
  case MONDAY:
    break;
  case TUESDAY:
    compute(args); // 将标记为for语句的内容放入指定的方法中
    break;

    /* ... */
}
