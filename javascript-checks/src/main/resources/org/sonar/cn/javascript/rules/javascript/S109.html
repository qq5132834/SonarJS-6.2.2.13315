在语句中使用不知从何而来的魔鬼数字。例如，经常使用魔鬼数字来限制循环的迭代次数，以测试属性的值。

当在编写一段代码时，使用魔鬼数字可能看起来很明显很简单，但在调试时，它们就不会变得莫名其妙。

这就是为什么在使用魔鬼数字之前，必须先将其分配给明确命名的变量，以便能解开其神秘面纱。

-1、0、1通常不被认为是魔鬼数字。

不合规示例

function doSomething() {
  for (let i = 0; i < 4; i++) {    // 不合规，4是魔鬼数字
    // ...
  }
}

合规示例

function doSomething() {
  const numberOfCycles = 4;
  for (let i = 0; i < numberOfCycles; i++) {
    // ...
  }
}
