使用等式运算符（==和!=）在循环判断终止中是危险的，因为它可能会导致死循环。 相反，使用关系运算符效果可能会更好，但并不绝对。

不合规示例

for (var i = 1; i != 10; i += 2)  // i != 10 不合规.
{
  //...
}

合规示例

for (var i = 1; i <= 10; i += 2)  // i <= 10 合规
{
  //...
}

例外

如果未在循环主体内修改循环计数器，则忽略相等运算符，并且：

  • 从结束值以下开始，每次迭代递增1。
  • 开始于结束值之上，每次迭代递减1。

当测试针对null时，也会忽略相等运算符。

for (var i = 0; arr[i] != null; i++) {
  // ...
}

for (var i = 0; (item = arr[i]) != null; i++) {
  // ...
}
