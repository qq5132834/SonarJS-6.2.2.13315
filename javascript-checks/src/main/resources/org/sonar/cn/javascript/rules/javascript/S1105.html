当大括号样式不遵守规范时，此规则会引发问题：

不合规示例

使用参数：

if (condition)
{                                                      //不合规
  doSomething();
}                                                      //不合规
else {
  doSomethingElse();
}{code}

合规示例

if (condition) {                                   //合规
  doSomething();
} else {                                           //合规
  doSomethingElse();
}

例外

* 作为参数出现时 

functionWithObject(
   {                                                 //合规
        g: "someValue"
   }
);

* 当块被内联时（左右大括号在同一行），不会触发任何问题。

if(condition) {doSomething();}                       //合规
