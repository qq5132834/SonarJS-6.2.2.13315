代码中使用大括号样式不遵守规范时，此规则会引发问题：

不合规示例

使用参数：

if (condition)
{                                                      //不合规
  doSomething();
}                                                      //不合规
else {
  doSomethingElse();
}{code}

合规示例

if (condition) {                                   //合规
  doSomething();
} else {                                           //合规
  doSomethingElse();
}
