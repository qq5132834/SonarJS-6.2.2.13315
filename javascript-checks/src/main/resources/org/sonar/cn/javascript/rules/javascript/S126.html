当 if 后跟多个 else if 语句时，最后一个 else if 建议改为 else 语句。

这是因为最后else语句的要求是防御性编程。

最后的 else 语句应该采取适当的操作，或者包含一个适当的注释，说明为什么不采取任何操作。这与在switch语句中 使用加入 default 语句一样。

不合规示例

if (x == 0) {
  doSomething();
} else if (x == 1) {
  doSomethingElse();
}

合规示例

if (x == 0) {
  doSomething();
} else if (x == 1) {
  doSomethingElse();
} else {
  throw "Unexpected value for x";
}
