集合的大小和数组的长度始终大于或等于零。 因此，测试大小或长度是否大于或等于零没有意义，因为结果始终为 true。 类似地，测试它是否小于零将始终返回 false。 也许目的是检查集合或数组的非空性。

不合规示例

if (someSet.size >= 0) {...} // 不合规

if (someMap.size < 0) {...} // 不合规

const result = someArray.length >= 0;  // 不合规

合规示例

if (someSet.size > 0) {...}

if (someMap.size == 0) {...}

const result = someArray.length > 0;
