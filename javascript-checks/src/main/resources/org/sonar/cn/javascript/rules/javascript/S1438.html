在 JavaScript 中，分号 (;) 作为语句分隔符是可选的， 但是省略分号可能会造成混淆，并导致意想不到的结果，因为在每行的末尾隐式插入了分号。

不合规示例

function fun() {
  return  // 不合规. ';' implicitly inserted at end of line
       5   // 不合规. ';' implicitly inserted at end of line
}
print(fun());  // 打印 "undefined", not "5"

合规示例

function fun() {
  return 5;
}
print(fun());
