控制流语句"if"、"for"、"while"、"switch"和"try"不应嵌套太深

这样的代码很难阅读、重构和维护。

不合规示例

默认阈值为3:

  if (condition1) {                  // 合规深度 1
    /* ... */
    if (condition2) {                // 合规深度 2
      /* ... */
      for(let i = 0; i < 10; i++) {  // 合规深度 3，不超过限制
        /* ... */
        if (condition4) {            // 不合规深度 4
          if (condition5) {          // 深度 5，超过限制，但仅在深度深度4时报告问题
            /* ... */
          }
          return;
        }
      }
    }
  }
